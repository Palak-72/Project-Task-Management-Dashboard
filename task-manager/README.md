Project & Task Management Dashboard

A modern and responsive Project & Task Management Dashboard built using React. This application helps users manage projects and tasks efficiently with real-time updates, analytics, and interactive UI components.

 Features

Dashboard
Total Projects overview
Total Tasks count
Completed & Pending tasks tracking
Dynamic cards with auto-update
Interactive Pie Chart (task distribution)

 Project Management
View all projects
Add new projects
Delete projects
Project details including name, description, and status
 Task Management
Add new tasks
Delete tasks
Track task status (Completed / Pending)
Real-time updates in dashboard

Search & Filter
Search tasks by title
Filter tasks by status (All / Completed / Pending)
 Tables
Recent Tasks table
Recent Projects table
Includes date, status, and actions
➕ Add Modal
Add Task / Project via popup form
Instant UI update after submission
🛠️ Tech Stack
Frontend: React.js (Hooks-based)
Styling: Tailwind CSS
State Management: Context API
Routing: React Router DOM
Charts: Recharts
Animations: Framer Motion
Icons: Lucide React
API: JSONPlaceholder (for mock data)

Installation & Setup

 Clone the repository
git clone https://github.com/your-username/task-manager.git
cd task-manager

 Install dependencies
npm install

 Run the project
npm run dev

 Usage
Click Add Button to create a new task or project
Use Search & Filter to find specific tasks
View analytics on the dashboard
Delete tasks/projects using action buttons
